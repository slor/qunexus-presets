qunexus-presets
===============

Preset files for the KMI's [QuNexus](http://www.keithmcmillen.com/qunexus/tour).

Preset files are JSON. I export them from the QuNexus editor then run them through [JSONLint](http://jsonlint.com/) for readability.

Korg iMS-20
===========

Korg iMS-20 Drums
-----------------

* Notes 0-5 control drum pads.
* Other notes control keyboard.
* Can adjust keyboard octaves and keep the drum pads in place.
